Tools File
----------

This file contains the class :class:`Tools <FPSDP.Diagnostics.BES.tools.Tools>` that contains some methods for the analysis of the BES pictures and
the file defines some function for checking the convergence, helping with the input file or creating some plots of the different quantities.
It can be a good start to understand how the code works.
