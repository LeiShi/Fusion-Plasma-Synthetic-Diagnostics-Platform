Collisions Directory
====================

This package contains all the physics for the collisions.
The class :class:`ADAS_file <FPSDP.Plasma.Collisions.ADAS_file.ADAS_file>` read the ADAS files and store them, they can be acces with the submodule :mod:`collisions <FPSDP.Plasma.Collisions.collisions>` that will interpolate the values.
