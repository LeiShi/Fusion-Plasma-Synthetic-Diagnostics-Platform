Tools File¶

This file contains the class Tools that contains some methods for the analysis of the BES pictures and the file defines some function for checking the convergence, helping with the input file or creating some plots of the different quantities. It can be a good start to understand how the code works.

class FPSDP.Diagnostics.BES.tools.Tools(I, R, Z, psin, name_id, I_fluc=False)[source]¶

Defines a few tools for doing some computations on the BES image

Two different methods of initialization have been made: The first one is the usual __init__ function that take as input the photon radiance, the R,Z coordinates and the psin value, and the second one read a npz file with a standard order and calls the __init__ method.

Parameters:
  • I (np.array[Nt,Nfib]) – Photon radiance
  • R (np.array[Nfib]) – R coordinate
  • Z (np.array[Nfib]) – Z coordinate
  • psin (np.array[Nfib]) – \(\Psi_n\) value
  • name_id (str) – Name defining the data (used for the titles))
  • I_fluc (bool) – Input are fluctuation or total intensity

FPSDP.Diagnostics.BES.tools.compute_scaling_factor(ne_fluc=0.1, T_fluc=0.01, radial=True, Radius=[1.67, 0.67], Nr=100, graph=True, xgc=None)[source]¶

Compute the scaling factor relating the density fluctuations to the BES fluctuations: \(C\frac{\tilde{I}}{I}=\frac{\tilde{n}_e}{n_e}\) This code is more or less a copy of the BES code, therefore it should be changed if the main code is changed

Parameters:
  • ne_fluc (float) – Ratio of density fluctations
  • T_fluc (float) – Ratio of temperature fluctuations
  • radial (bool) – Choice between real focus point or a radial analysis (on the midplane)
  • Radius (list[float,float]) – Major and minor radius of the tokamak (default is D3D, useful only if radial is True)
  • Nr (int) – Number of fiber (useful only if radial is True)
  • graph (bool) – Choice bwteen return the values or ploting the graph
  • xgc (load_XGC_local) – Choice between loading a new XGC or using an existing one.
Returns:

Position (R or (R,Z)), the coefficient C and the difference between I_fl and I
Return type:

(np.array[Nr])*3 or (np.array[Nr])*4
