Beam Emission Spectroscopy Documentation¶

The Beam Emission Spectroscopy (BES) synthetic diagnostic use a configuration file (e.g. bes.in). The only object that should be used for the diagnostic is the class BES, it creates everything from the config file and with the method get_bes() will compute the number of photons captured by each fiber. example.py is an example of how to use the code.

In the documentation, I tried to specify the unit of each variable, but it is possible that I forgot a few. I choose to use the SI system as often as possible (except for the beam energy), therefore it should be the first guess.

FPSDP Directory¶

The main directory of the library contains the following directory:

  • Diagnostics Directory contains the files for the beam and the BES optics.
  • Input and Output Directory contains a function for parsing a string.
  • Maths Directory contains the quadratures points and weights.
  • Collisions and Plasma Simulations Directory contains the files about the collisions and the output of the simulations.
  • General Settings Directory contains the definition of a few constants.
