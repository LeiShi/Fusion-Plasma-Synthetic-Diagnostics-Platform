Collisions Directory
====================

This package contains all the physics for the collisions.
:mod:`FPSDP.Plasma.Collisions.ADAS_file` read the ADAS files and store them, they can be acces with the submodule :mod:`FPSDP.Plasma.Collisions.collisions` that will interpolate the values.
